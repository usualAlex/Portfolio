ArmorCraft







PLATFORM: Android / iOS
ENGINE: Unity
GENRE: Strategy / Base Building / F2P / MMO
DEVELOPER / PUBLISHER: xFuture Technology Co.
STATUS: Canceled
MY ROLE: UI/UX Designer


A canceled game i worked on UI for 3 years.

ArmorCraft was a strategy game - base building, alliance warfare, PvP and PvE. Unfortunately, the game has been canceled after the data collected from BETA testing was not good enough.

A complex and challenging UI. I had to design a simple to use UI with a lot of information required to display on a tiny phone screen.

The following are UI screens that I designed for the game. Many elements were still left unfinished or in a WIP state.



Homebase Latest UI

Exploring different UI styles early on in relation to the changes to the art direction of the game. It was a lot of work having to redesign the UI from scratch several times. From military themes to a more sci fi punk style.
